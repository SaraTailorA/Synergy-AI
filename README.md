## About the Project

**SynergyIA** is a professional multi-page website for an AI consulting company that helps businesses implement Artificial Intelligence solutions in **30 days or less**. The site showcases services, team members, company values, and provides a contact form for potential clients.

This project was built using **only HTML5 and CSS3**, with no JavaScript frameworks or libraries, making it an ideal example of clean, semantic, and accessible web development.

---

## Pages

| Home | `index.html` | Landing page with hero section, benefits, and value proposition |
| Who We Are | `who_we_are.html` | Mission, vision, values, and team members |
| Services | `services.html` | AI services offered, industries served, and FAQ |
| Contact | `contact.html` | Contact form, company info, and Google Maps location |

---

## Technologies Used

| Technology | Usage |
| HTML5 | Semantic page structure |
| CSS3 | Styling, layout, animations, and responsive design |
| **CSS Grid** | Responsive grid layouts for cards and services |
| **CSS Flexbox** | Header navigation and component alignment |
| **CSS Variables** | Centralized color palette and design tokens |
| **Google Maps Embed** | Interactive map on the contact page |

## Features

### Design
- [x] Clean, modern, Apple-inspired design
- [x] Consistent color palette using CSS custom properties
- [x] Smooth hover animations and transitions
- [x] Glass-effect sticky header with backdrop blur
- [x] Pill-shaped buttons with hover scaling effect
- [x] Card components with shadow and levitation on hover


### Accessibility
- [x] Semantic HTML5 elements (`<header>`, `<main>`, `<nav>`, `<article>`, `<footer>`)
- [x] Descriptive `alt` attributes on all images
- [x] Proper heading hierarchy (h1 → h2 → h3)
- [x] `<label>` elements linked to form inputs
- [x] High contrast text colors
- [x] Meaningful link text

### Forms
- [x] Contact form with text, email, select, and textarea fields
- [x] Custom styled inputs with focus states
- [x] Placeholder text for user guidance
- [x] Required field validation

### Other
- [x] FAQ section using `<details>` and `<summary>` elements
- [x] Embedded Google Maps iframe
- [x] Active page indicator in navigation
- [x] Animated underline effect on nav links

## How to Run

1. Clone this repository or download the project files.
2. Ensure all files (`index.html`, `who_we_are.html`,`services.html`,`contact.html`,`styles.css`) are in the same folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
